#### Exercise - Array

**By using the following array:**
```javascript
let arr_1 = ["Moh", "Jouza", "Swift", "Tuwaiq-1000"]
```
- print second object and forth one.
```javascript
arr_1[1]
arr_1[3]
```
- change third object to JavaScript.
```javascript
arr_1[2] = "Javascript"
```
- print the length of the array.
```javascript
arr_1.length
```
